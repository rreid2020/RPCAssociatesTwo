# Quick Docker Commands Reference

## Local Development

### Build the image
```bash
docker build -t rpc-associates .
```

### Run the container
```bash
docker run -p 80:80 rpc-associates
```

### Build and run with docker-compose
```bash
docker-compose up -d --build
```

### View logs
```bash
docker-compose logs -f
```

### Stop the container
```bash
docker-compose down
```

### Rebuild after changes
```bash
docker-compose up -d --build
```

## Testing Locally

1. Build: `docker build -t rpc-associates .`
2. Run: `docker run -p 8080:80 rpc-associates`
3. Visit: `http://localhost:8080`
