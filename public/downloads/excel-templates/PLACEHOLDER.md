# Excel Templates Directory

Place your Excel template files here.

## Cash Flow Statement Template

**File name:** `cash-flow-statement-template.xlsx`

**Location:** This file should be placed in this directory.

**URL:** Once deployed, the file will be available at:
`https://rpcassociates.co/downloads/excel-templates/cash-flow-statement-template.xlsx`

## Adding More Templates

1. Place `.xlsx` files in this directory
2. Update `src/pages/Resources.tsx` to add resource entries
3. Create dedicated pages in `src/pages/` if needed
4. Add routes in `src/App.tsx`
